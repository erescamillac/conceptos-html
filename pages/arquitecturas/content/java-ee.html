Java EE

Creado por Erick Escamilla C.

Java Platform, Enterprise Edition (Java EE) es el estándar del software empresarial impulsado por la comunidad. Java EE se desarrolla mediante el Proceso de la Comunidad Java, con contribuciones de expertos del sector, organizaciones comerciales y de código abierto, grupos de usuarios de Java e innumerables personas. Cada versión integra nuevas funciones que se ajustan a las necesidades del sector, mejora la portabilidad de las aplicaciones y aumenta la productividad de los desarrolladores.

Java EE 8 sigue mejorando las API y los modelos de programación necesarios para las aplicaciones actuales y añade funciones solicitadas por nuestra comunidad mundial. Esta versión moderniza la compatibilidad con muchos estándares del sector y continúa la simplificación de las API preparadas para la empresa. Las mejoras incluyen:

  • API Java Servlet 4.0 con soporte HTTP/2
  • Compatibilidad mejorada con JSON, incluida una nueva API de enlace JSON
  • Una nueva API de cliente reactivo REST
  • Eventos CDI asíncronos
  • Una nueva API de seguridad portátil
  • Soporte de eventos enviados por el servidor (del lado del cliente y del servidor)
  • Soporte para las nuevas capacidades de Java SE 8 (por ejemplo, API de fecha y hora, API de flujos, mejoras en las anotaciones)
