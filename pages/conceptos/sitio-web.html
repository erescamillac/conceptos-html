Sitios Web

Creado por Erick Escamilla C.

Un sitio web es un conjunto de páginas web interconectadas y de acceso público que comparten un mismo nombre de dominio. Los sitios web pueden ser creados y mantenidos por un individuo, un grupo, una empresa o una organización para servir a una variedad de propósitos.

10 componentes esenciales para el éxito de un sitio web

  1. Un sistema de gestión de contenidos (CMS)
  2. Un servicio de alojamiento (hosting) web fiable
  3. UX (experiencia de usuario)
  4. Conceptos básicos de SEO (Search Engine Optimization)
  5. Herramientas de análisis
  6. Propuesta de valor
  7. Página de testimonios
  8. Campañas de correo electrónico
  9. Formularios de contacto con páginas de agradecimiento
  10. Enlaces e integración en las redes sociales
