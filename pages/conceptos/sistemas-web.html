Sistemas Web

Creado por Erick Escamilla C.

El sistema de información web, o sistema de información basado en la web, es un sistema de información que utiliza las tecnologías web de Internet para ofrecer información y servicios, a los usuarios o a otros sistemas de información/aplicaciones. Se trata de un sistema de software cuyo objetivo principal es publicar y mantener datos utilizando principios basados en el hipertexto.

Arquitectura de los sistemas basados en la web

Arquitecturas cliente-servidor y web de 2 niveles

Normalmente, cuando se navega por Internet, se utiliza un navegador web como Internet Explorer o Mozilla Firefox. El ordenador que ejecuta un navegador se llama cliente, mientras que la máquina que proporciona las páginas web se llama servidor.

Cuando usted se conecta a un proveedor de servicios de Internet (ISP), por ejemplo, Blue Yonder o Aol, su ordenador establece una conexión de red con un servidor web. En esta situación, su ordenador es en efecto un cliente, que está conectado a un servidor web del ISP. El servidor web, como su nombre indica, sirve a su navegador páginas web (por ejemplo, páginas HTML, ASPX, JSP, etc.).

Este sencillo escenario, en el que el servidor web está conectado a uno o varios clientes, se conoce como modelo de arquitectura de 2 niveles.

Arquitecturas de 3 niveles

En general, las aplicaciones informáticas constan de tres tipos de funcionalidades distintas y diferenciadas.

  • Servicios de presentación: Se manifiestan en forma de visualización de información y facilidades para la introducción de datos por parte del usuario. Por lo general, es el front-end para la interacción con el usuario. Por ejemplo, el inicio de sesión requiere una interacción en forma de recogida de información sobre el nombre de usuario y la contraseña mediante un formulario HTML.
  • Lógica funcional: Toda aplicación incluye algún tipo de procesamiento de datos y esto también puede implicar la interactividad con la base de datos. Por ejemplo, la autenticación de usuarios requiere que la unidad lógica lea las combinaciones de nombre de usuario y contraseña de una base de datos y las compare hasta llegar a una buena comparación (con suerte).
  • Gestión de datos: Los datos, su almacenamiento, inserción y recuperación, su gestión y alteración son fundamentales para las aplicaciones informáticas. Por ejemplo, un sistema de gestión de bases de datos (DBMS) es necesario para la gestión de los nombres de usuario y las contraseñas asociadas, sus propietarios, etc.
