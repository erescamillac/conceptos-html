¿Qué es un Servidor de desarrollo?

Creado por Erick Escamilla C.

Un servidor de desarrollo es un tipo de servidor que está diseñado para facilitar el desarrollo y la prueba de programas, sitios web, software o aplicaciones para los programadores de software. Proporciona un entorno de ejecución, así como todas las utilidades de hardware/software que son esenciales para la depuración y el desarrollo de programas.

Servidor de Aplicación

Los servidores de aplicaciones proporcionan un marco para construir y desplegar aplicaciones web y ofrecen una variedad de servicios cuando se ejecutan dichas aplicaciones. Estos servicios incluyen seguridad, transacciones, clustering para aumentar el rendimiento y capacidades de diagnóstico, entre otros. Estos productos también pueden incluir servidores que alojan estrictamente aplicaciones web. Los servidores de aplicaciones son utilizados por los desarrolladores de software que intentan crear rápidamente aplicaciones y que éstas sean soportadas por el entorno del servidor en el que se despliegan. Estos productos permiten a los desarrolladores centrarse en la propia aplicación y dedicar menos tiempo a cuestiones externas, como el rendimiento o la seguridad. Los servidores de aplicaciones pueden funcionar junto con bases de datos relacionales y una variedad de fameworks web.
