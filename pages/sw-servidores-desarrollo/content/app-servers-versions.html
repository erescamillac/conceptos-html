IBM WebSphere Application Server

IBM WebSphere Application Server es el entorno de ejecución basado en Java™ Enterprise Edition (Java EE) más fiable, seguro y flexible. IBM WebSphere® abre la puerta a cualquier desarrollador que utilice cualquier herramienta para cualquier aplicación, en cualquier nube. WebSphere es un entorno de servidor altamente escalable diseñado para alojar aplicaciones en la nube y microservicios en organizaciones de cualquier tamaño. Aumente la productividad de los desarrolladores en un 25 por ciento con WebSphere Liberty, un tiempo de ejecución que se incluye en todas las versiones, gracias a la migración cero de su infraestructura de aplicaciones y a la posibilidad de acceder a nueva capacidad y mover aplicaciones a la nube en minutos. Reduzca los costes mediante el uso de WebSphere Liberty gracias a una administración simplificada y a una reducción del 30 por ciento en la utilización de la infraestructura. Versiones del producto 1. WebSphere Application Server: Utilice esta versión para aplicaciones Java EE Full Profile que necesiten integrarse con aplicaciones Java EE existentes, como las que utilizan JMS. 2. 2. WebSphere Liberty Core: La versión ideal para aplicaciones web o microservicios que dependen únicamente del Java EE Web Profile. 3. WebSphere Application Server ND: Utilice esta versión para gestionar aplicaciones Java EE Full Profile que se ejecutan fuera de un entorno de contenedores. 4. Edición familiar de WebSphere Application Server: La edición Family ofrece una mayor flexibilidad de licencias en su empresa para tres productos WebSphere: WebSphere Application Server, WebSphere Liberty Core y WebSphere Application Server ND.
