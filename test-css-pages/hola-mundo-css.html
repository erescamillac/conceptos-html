Aplicación Web VS. Sitio web
Parámetro Aplcación Web Sitio Web
Creado para Una aplicación web está diseñada para interactuar con el usuario final Un sitio web se compone principalmente de contenido estático. Es de acceso público para todos los visitantes.
Interacción con el usuario En una aplicación web, el usuario no sólo lee el contenido de la página, sino que también manipula los datos restringidos. Un sitio web ofrece contenidos visuales y de texto que el usuario puede ver y leer, pero que no afectan a su funcionamiento.
Autenticación Las aplicaciones web necesitan autenticación, ya que ofrecen un abanico de opciones mucho más amplio que los sitios web. La autentificación no es obligatoria para los sitios web informativos. El usuario puede pedir registrarse para obtener una actualización periódica o para acceder a opciones adicionales. Estas funciones no están disponibles para los visitantes del sitio web no registrados.
Tarea y complejidad Las funciones de las aplicaciones web son bastante más altas y complejas en comparación con un sitio web. El sitio web muestra los datos y la información recogidos en una página específica.
